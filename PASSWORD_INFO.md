# 密码加密说明

## 当前状态

系统已启用BCrypt密码加密功能。

## 测试账号

- **用户名**: admin
- **密码**: admin123
- **角色**: ADMIN

## 密码加密机制

1. **登录验证**: 使用BCrypt比对明文密码和数据库中的加密密码
2. **新增用户**: 自动加密密码后存入数据库
3. **修改密码**: 自动加密新密码后更新数据库
4. **编辑用户**: 如果不修改密码，保持原加密密码不变

## 数据库密码格式

- 加密后的密码格式: `$2a$10$...` (60个字符)
- 示例: `$2a$10$N9qo8uLOickgx2ZMRZoMyeIjZAgcfl7p92ldGxad68LJZdL17lhWy`

## 注意事项

1. 数据库中现有的明文密码需要手动更新为加密格式
2. 新创建的用户密码会自动加密
3. BCrypt每次加密同一密码会生成不同的哈希值（包含随机盐值）
4. 密码验证使用BCrypt的matches方法，会自动处理盐值

## Token有效期

- 当前设置: 10秒（测试用）
- 生产环境建议: 5-30分钟
- 配置位置: `backend/src/main/java/com/example/chartboardbackend/util/JwtUtil.java`
